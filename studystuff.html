Operators in Python

Python offers a wide range of operators, each serving a specific purpose:

  • Arithmetic: +, -, *, /, %, **, //
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not
  • Assignment: =, +=, -=, *=, etc.
  • Bitwise: &, |, ^, ~, <<, >>
  • Membership: in, not in
  • Identity: is, is not
Example: x = 5; y = 3; print(x ** y) ‚Üí Output: 125

These operators are foundational for building logic, calculations, and control flows in any Python application‚Äîfrom microbusiness dashboards to AI models.

Operator Precedence

Python evaluates expressions based on a strict hierarchy:

  1. Parentheses ()
  2. Exponentiation **
  3. Unary +, -
  4. Multiplication, Division, Floor, Modulus
  5. Addition, Subtraction
  6. Comparisons
  7. Logical NOT, AND, OR
Example: print(3 + 4 * 2) ‚Üí Output: 11 (not 14!)

Understanding precedence is crucial for writing predictable, bug-free code‚Äîespecially in financial calculations or inventory logic.

Print Formatting

Python supports multiple formatting styles:

  • Concatenation: print("Hello", name)
  • f-Strings: print(f"Hello {name}")
  • Format Method: "{:,.2f}".format(12345.678)
  • Percent Style: "%s scored %d" % ("Ali", 95)
Example: print(f"Total: PKR {amount:,.0f}") ‚Üí Output: Total: PKR 125,000

Used in dashboards, receipts, logs, and user interfaces‚Äîprint formatting ensures clarity and professionalism.
