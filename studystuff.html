Python Operators

Python supports several types of operators:

  • Arithmetic: +, -, *, /, %, **, //
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not
  • Assignment: =, +=, -=, *=, etc.
  • Bitwise: &, |, ^, ~, <<, >>
  • Membership: in, not in
  • Identity: is, is not
Example: x = 5; y = 3; print(x ** y) ‚Üí Output: 125

Operator Precedence

Python evaluates expressions based on precedence:

  1. Parentheses ()
  2. Exponentiation **
  3. Unary +, -
  4. Multiplication, Division, Floor, Modulus
  5. Addition, Subtraction
  6. Comparisons
  7. Logical NOT, AND, OR
Example: print(3 + 4 * 2) ‚Üí Output: 11 (not 14!)

Print Formatting

Python offers multiple ways to format output:

  • print("Hello", name)
  • print(f"Hello {name}") ‚Üê f-strings
  • "{:,.2f}".format(12345.678) ‚Üí 12,345.68
  • print("%s scored %d" % ("Ali", 95))
Use f-strings for clean, readable formatting in modern Python.
