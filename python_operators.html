1. Arithmetic Operators

  • + Addition: 3 + 2 = 5
  • - Subtraction: 5 - 2 = 3
  • * Multiplication: 4 * 2 = 8
  • / Division: 10 / 2 = 5.0
  • % Modulus: 10 % 3 = 1
  • ** Exponent: 2 ** 3 = 8
  • // Floor Division: 10 // 3 = 3

2. Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

3. Logical Operators

  • and Returns True if both statements are true
  • or Returns True if one statement is true
  • not Reverses the result

4. Assignment Operators

  • = Assign value
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign

5. Membership & Identity Operators

  • in Checks if value is in a sequence
  • not in Checks if value is not in a sequence
