📦 Smart Replenishment Tool

Predict. Prioritize. Prevent stockouts — intelligently.

🔧 What It Does

  • Tracks stock levels and dispatches
  • Forecasts depletion based on lead time
  • Triggers low-stock alerts
  • Exports urgency reports to Excel
  • Simulates multi-day warehouse operations
View GitHub Repo →

📁 How to Use

Clone the repo and run the CLI tool:

git clone https://github.com/emadaed/smart-replenishment-tool  
cd smart-replenishment-tool  
python replenish.py

Follow the prompts to enter stock levels, lead times, and dispatch quantities. The tool will generate an Excel report with urgency flags and daily sheets.

📊 Sample Output

Excel file with:

  • Urgency column (color-coded)
  • Daily simulation sheets
  • Dispatch validation logic
